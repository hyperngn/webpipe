WebPipe | Pipe your logs to the web!

Just use the following code to start piping data to your own private URL! 

tail -f access.log | curl --upload-file - https://6a625a69.ngrok.io/session/toronto-elixir-is-awesome-62ed73fdd2
or to send a message use: 
echo "Danny says Hi!" | curl --upload-file - https://6a625a69.ngrok.io/session/toronto-elixir-is-awesome-62ed73fdd2
to view the incoming messages in the browser go to: https://6a625a69.ngrok.io/session/toronto-elixir-is-awesome-62ed73fdd2
